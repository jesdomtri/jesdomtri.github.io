Experiencia Laboral

Analista y desarrollador

Viajes El Corte Inglés | Octubre 2021 - Actualidad


Durante mi empleo actual, he trabajado extensamente con tecnologías .NET, desde la versión framework 4.5.2 hasta la más reciente, la versión 8, en el desarrollo de diversas soluciones, tales como APIs REST, microservicios y workers. En todos mis proyectos, me he enfocado en la implementación de buenas prácticas de programación, incluyendo principios SOLID, y he utilizado herramientas de control de versiones, como Azure y TFS. También he utilizado Azure DevOps para la implementación y gestión de procesos de Integración y Despliegue Continuos (CI/CD), incluyendo la creación y administración de releases y pipelines. Por otro lado, he utilizado herramientas de evaluación de calidad de código, como SonarCloud, y he implementado tests unitarios e integración para garantizar la calidad del software. Por último, mencionar el uso de Swagger para proporcionar una interfaz de usuario sencilla.

De entre todos los proyectos en los que he trabajado cabe destacar:


  • API REST que conecta SAP con la API de gestión de cierre de reservas, y contiene todas las funcionalidades financieras.
  • API REST encargada de todos los procesos necesarios en el cierre de una reserva.
  • Worker para la automatización de tareas financieras y administrativas.
  • Worker que automatiza la generación de informes financieros trimestrales para consumo de contabilidad.
  • API REST para la gestión de recursos administrativos.
  • API REST que unifica la información que se obtiene desde dos APIs distintas.
  • Intranet desarrollada con Angular 14 para front y .NET para back para la gestión de sociedades.
  • Intranet desarrollada con Angular 14 para front y .NET para back enfocada en la gestión y visualización de dashboards de Looker y PowerBI.

Respecto a la gestión de tareas, he seguido la metodología SCRUM para la gestión de tareas en proyectos de equipo, lo que ha optimizado la eficiencia en la ejecución de proyectos y facilitado una comunicación clara y efectiva entre los miembros del equipo. Asimismo, he utilizado el sistema Kanban para el seguimiento de tareas e incidencias durante las reuniones diarias, asegurando un control preciso del progreso y resolución oportuna de problemas.
